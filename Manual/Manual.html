🔧 Required Installation Steps (Must do before opening the program)

Part 1: Tesseract OCR (Required - for reading text from images)

This program requires Tesseract to read text from the screen area you select

  1. Download: Go to https://github.com/UB-Mannheim/tesseract/wiki
  2. Select the installer file named tesseract-ocr-w64-setup-v5.x.x.exe (choose the latest version)
  3. Install: Double-click the downloaded file to install
  4. ⚠️ Warning: During installation, absolutely do not change the installation location. It must be installed at C:\Program Files\Tesseract-OCR\ only

Part 2: Ollama AI Platform (Required - for running AI Model for translation)

This program uses Ollama to run AI Models that handle translation

  1. Download: Go to https://ollama.com/download and select download for Windows
  2. Install: Double-click the downloaded file to install. When finished, restart your computer once
  3. Download AI Model:
    • Press the Start button and type cmd. You will see a program called "Command Prompt". Click to open it
    • In the black window that opens, copy and paste the command below, then press Enter (recommended to install Gemma 3n first because of good and accurate translation quality)
    ollama pull gemma3n
    • Wait until the download is complete (may take a while). When finished, it will show Success

Once you complete both parts, you're ready to use the Screen Translator program!

🚀 Opening and Using the Program

  1. Go back to the program folder and double-click ScreenTranslator.exe to open
  2. A small dark control window will appear
  3. Click the "📸 Capture Screen" button or press the Alt key on the keyboard twice quickly (Alt+Alt) to start capturing the screen
  4. Drag the mouse to select the text area you want to translate
  5. When you release the mouse, the translation result window will appear with the translation

Important note: Do not delete or move the folder named _internal as it contains important program files

📥 How to Install Additional Models:

Open Command Prompt and use the following commands according to the Model you want

Note: Downloading each Model will take some time depending on your internet speed. Please be patient until you see "success" or "pulling manifest" message
  • Gemma 3n (Already recommended) 
    ollama pull gemma3n

    Size: ~2.0GB, good translation quality, accurate, suitable for general translation work

  • Gemma 3n (Unsloth) (RAM-efficient) 
    ollama pull hf.co/unsloth/gemma-3n-E4B-it-GGUF:Q4_K_XL

    Size: ~1.8GB, optimized version for RAM efficiency, suitable for older machines

⚙️ How to Change Model in the Program:

  1. In the translation result window, click on the gear icon (⚙️)
  2. Select the Model you want from the list
  3. Click "💾 Save Settings" to save

🗑️ Managing Models (Deleting Unwanted Models):

If the installed AI Models take up too much disk space, you can remove unused Models

Tip: Each AI Model takes about 1.5-3GB of space. It's recommended to keep only 1-2 Models that you actually use

📋 How to View List of Installed Models:

ollama list

This command will show all Models installed on your machine

🗑️ How to Delete Unwanted Model:

Use the command below, replacing [MODEL_NAME] with the name of the Model you want to delete:

ollama rm [MODEL_NAME]

Examples of deleting Models:

  • Delete gemma3n: ollama rm gemma3n
  • Delete Gemma 3n Unsloth: ollama rm hf.co/unsloth/gemma-3n-E4B-it-GGUF:Q4_K_XL
⚠️ Warning: Once you delete a Model, you'll need to download it again if you want to use it. So only delete Models you're sure you won't use

💻 System Requirements

Minimum:

  • Windows 10 64-bit or newer
  • 8GB RAM
  • 10GB free space

Recommended:

  • 16GB+ RAM
  • NVIDIA RTX 3060 graphics card or higher (will help processing speed significantly)
  • Use SSD for data storage

⚠️ Important Warning: When Opening the Program for the First Time (Windows SmartScreen)

When you open ScreenTranslator.exe for the first time, Windows may display a blue window to protect against unknown programs. This is normal for programs that are not yet widely recognized.

Easiest way to open the program:

  1. When the "Windows protected your PC" window appears, click on the "More info" text
  2. Then you will see the "Run anyway" button appear. Click this button to open the program
  3. Generally, you only need to do this once for the first-time setup

🛠️ Common Troubleshooting

  • Shows "Connection Refused": Try opening Command Prompt and run ollama serve command, then open the program again
  • Can't translate / OCR can't read: Check if Tesseract OCR is properly installed at C:\Program Files\Tesseract-OCR\
  • Program runs slowly: Try switching to a smaller AI Model like Gemma 3n (Unsloth)
  • Not enough RAM: Close other unnecessary programs while using
